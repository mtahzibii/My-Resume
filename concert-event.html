Concert Events, a DJ Event App

  • Source Code
    •
App Review

This application is a platform built with Next.js as front-end and Strapi as backend. In this application, user can register and create an event (e.g. concert).
To manage concert data, Strapi v.4 is being used which plays as an Headless CMS. Also to manage media, cloudinary as media API is being used.

Other libraries used in the project:
  • JWT (Authentication)
  • React-Toastify (Notifications)
