App Review

This is a Wildfire Tracker application which displays the exact location of current wildfires around the world. The data is received through NASA API and displayed through map which is provided by Google Map API.
The main libraries used in the project are given below:

  • Axios (CRUD Operations)
  • React Leaflet (React components for Leaflet maps )
  • React-Bootstrap (UI framework, Rebuilt for React)
