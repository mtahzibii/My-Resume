App Review

This is an house marketplace application where users can sell or rent their house. After logged in, the user can view the list of houses registered for sale or rent. Also, if you wish, you can sell or rent a specific apartment. At the time of registration, it specifies the main specifications, including the toilet, bathroom, rooms, parking,fuenished, sale or rent, as well as the exact location of the apartment.
The backend of this application is powerved by Firebase which is a platform developed by Google for creating mobile and web applicationsis. All is required in backend (including Database, Data Store, Authentication and ...) are being easily provided by this platform.

Main libraries used in the project:
  • React Leaflet (React components for Leaflet maps )
  • Swiper (Touch Slider)
  • React-Toastify (Notifications)
