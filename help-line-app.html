App Review

This is a help desk application which is built with MERN (MongoDB, Express, React, Node.js) stack. In this application, user can register and create an acount to make a request to help center.
This application makes it possible for user to be able to register a report for one of Apple's products, including iPhone, Macbook Pro, iMack and iPad. It is possible to communicate between the customer and the support center in the form of sending messages for each report. After resolving the issue, the report will be colsed. The database is powerved by MongoDB Atlas Database which is a cloud-based platform. The users are being authenticated by JSON Web Token (JWT).

The main libraries used in the project are given below:

  • Axios (CRUD Operations)
  • Redux Toolkit (State Management)
  • JSON Web Token (Authentication)
  • React-Bootstrap (UI framework, Rebuilt for React)
  • React-Toastify (Notifications)
