App Review

This portfolio application was built using cutting-edge front-end technologies such as React, React-hook-form, react-icons, react-tooltip, and Framer Motion.
The back-end is powered by Sanity, a powerful content management system that allows for flexible content modeling and easy integration with front-end frameworks.
The portfolio app features a clean and modern design, with smooth animations and interactive components that provide an engaging user experience. It allows you to showcase your work, skills, and experience in a visually appealing and easy-to-navigate format.
One of the key features of the portfolio app is the use of React-hook-form for form validation, which provides a seamless and intuitive user experience. Additionally, react-icons and react-tooltip were used to add visual interest and interactivity to the UI.

Core web technologies integrated in the project:
  • React (Front-end library)
  • Framer-Motion (A motion library for React)
  • React-hook-form (Handle form and validation)
  • react-icons (popular icons in React projects)
  • Sanity.io (the platform for structured content)
  • react-tooltip (a tooltip component)
