App Review

This application is an e-Commerce platform built with MERN (MongoDB, Express, React, Node.js) stack. In this application, user can register and create an acount to place an order to buy site products.
Admin panel has been implemented to manage users, products and orders. By this panel, the admin user is able to delete and edit users, create and delete products, and also manage the status of orders. The database is powerved by MongoDB Atlas Database which is a cloud-based platform. The users are being authenticated by JWT. It is also provided to set comment for each product by users.

Main libraries used in the project:
  • Axios (CRUD Operations)
  • Redux Toolkit (State Management)
  • JSON Web Token (Authentication)
  • React-Bootstrap (UI framework, Rebuilt for React)
  • React-Toastify (Notifications)
