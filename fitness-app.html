App Review

This fitness application was built using the latest front-end technologies such as React.js and Material UI, resulting in a sleek and modern design with a user-friendly interface.
The app features a comprehensive catalog of exercises for all parts of the body, with pagination and a horizontal scroll bar for easy navigation and browsing. Designed to cater to all fitness levels and goals, this app provides a range of exercises to help users achieve their desired results.
Whether you're a beginner or an experienced fitness enthusiast, you'll find a variety of workouts to suit your needs.

The use of React.js allows for a seamless and responsive user experience, while Material UI enhances the app's visual appeal and usability. The app's pagination and horizontal scroll bar make it easy to find and access the exercises you need, with minimal effort.

Core web technologies integrated in the project:
  • React.js (Front-end library)
  • Material UI (UI Framework)
  • Scrolling horizontal menu component
