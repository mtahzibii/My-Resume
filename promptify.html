App Review

Promptify is a modern web application built using Next.js 13 and MongoDB for the database. It features a clean and user-friendly interface powered by tailwind CSS that allows you to create, edit, and delete prompts with ease. With the ability to add tags to each prompt. Promptify leverages the powerful Mongoose library to connect seamlessly with MongoDB, allowing for efficient data management and easy scalability. Promptify also uses NextAuth for client-side authentication, providing a secure and reliable way for users to sign in and access their prompts.The app provides a smooth user experience, with intuitive controls and responsive design that works across all devices. Overall, if you're looking for a reliable, user-friendly, and feature-rich web application for managing your prompts, Promptify is the perfect choice.

Core web technologies integrated in the project:
  • Next.js (React-based web application framework)
  • MongoDB (a tooltip component)
  • Tailwind CSS (CSS Framework)
  • NextAuth.js (Client side authentication for Next.js )
