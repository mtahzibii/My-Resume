Digital Headpones, an e-commerce App

  • Live Demo
    •
  • Source Code
    •
App Review

Developed a digital headphone e-Commerce platform using Next.js and Sanity as the headless CMS. Users can browse through available headphones, select desired items and add them to the cart for purchase. Payments were securely processed through Stripe API. Data storage and management is powered by Sanity CMS, allowing for efficient retrieval through API calls.

Main libraries used in the project:
  • Stripe (Payment)
  • canvas-confetti (drawing confetti on a canvas)
  • Next (React framework)
  • react-hot-toast (Notifications)
  • react-icons (Icons)
